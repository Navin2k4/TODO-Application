# Todo Application

This is a simple Todo Application built using **Vite** and **React**, with styling handled by **Tailwind CSS**. The project also includes **PostCSS** and **Autoprefixer** for processing CSS.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v14.18.0 or higher)
- [npm](https://www.npmjs.com/) (v6.0.0 or higher)
